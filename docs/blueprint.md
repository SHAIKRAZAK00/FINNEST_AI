# **App Name**: FinNest AI

## Core Features:

- Biometric Authentication: Secure app access using fingerprint or Face ID with PIN fallback.
- Family Management: Create or join families using a unique code for secure data sharing.
- Expense Tracking: Log expenses with categories, amounts, and contributors, synced in real-time.
- Goal Setting: Establish family financial goals and track progress collectively.
- AI Financial Assistant: Analyze spending habits and generate insights using generative AI to recommend optimized financial habits and find potential overspending as a tool.
- Gamified Savings: Earn points and badges for financial activities, with leaderboards for added motivation.
- Role Based Permissions: Set role-based Firebase read and write access controls for Parents, Children and Viewers.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to convey sophistication and financial awareness.
- Background color: Light gray (#E7E5E9), providing a clean and modern backdrop.
- Accent color: Muted lavender (#D0BCFF) as a highlight for interactive elements and calls to action.
- Body text: 'Inter', a sans-serif font providing clarity for financial details; Headline: 'Space Grotesk', for headings, leveraging a contemporary high-tech aesthetic.
- Use flat, minimalist icons with a touch of rounded edges for a friendly interface.
- Card-based design to organize and highlight key financial information, optimizing for different screen sizes.
- Smooth transitions and confetti effects to enhance user experience and celebrate achievements.
